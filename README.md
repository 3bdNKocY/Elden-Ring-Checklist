## Getting Started

### Prerequisites

- [npm](https://nodejs.org/en/download/package-manager/)

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Gobluebro/Elden-Ring-Checklist.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
